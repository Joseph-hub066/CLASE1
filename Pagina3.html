1. HTML

¿Que es HTML y para que sirve?

HTML (HyperText Markup Language) es el lenguaje de marcado estándar utilizado para crear y estructurar páginas web. Se utiliza para definir la estructura de contenido en la web mediante una serie de "etiquetas" o "marcadores" que indican cómo se debe organizar y mostrar el contenido en un navegador.

¿Para que sirve?

1. Estructuración de contenido: HTML permite organizar y estructurar diferentes tipos de contenido en una página web, como texto, imágenes, videos, tablas, enlaces, formularios y más.

2. Crear la base de las páginas web: Es el componente fundamental para crear cualquier página web, ya que sin HTML no habría una estructura base que los navegadores puedan interpretar y mostrar.

3. Interacción con otros lenguajes: HTML trabaja junto con otros lenguajes como CSS (para el diseño visual) y JavaScript (para la interactividad), pero el HTML sigue siendo el esqueleto básico de la página.

4. Accesibilidad y SEO: A través de una correcta estructuración en HTML, se pueden mejorar aspectos como la accesibilidad (para usuarios con discapacidades) y la optimización para motores de búsqueda (SEO), lo que puede ayudar a mejorar la visibilidad de un sitio web en los resultados de búsqueda.



2. CSS

CSS funciona como complemento a la información que forma parte de un sitio web. Mientras que el código en HTML incluye todos los datos, el código en CSS se encarga de darles formato y presentarlos visualmente a través de un navegador

3. JavaScript

JavaScript es un lenguaje de programación o de secuencias de comandos que te permite implementar funciones complejas en páginas web, cada vez que una página web hace algo más que sentarse allí y mostrar información estática para que la veas, muestra oportunas actualizaciones de contenido, mapas interactivos, animación de Gráficos 2D/3D, desplazamiento de máquinas reproductoras de vídeo, etc., puedes apostar que probablemente JavaScript está involucrado. Es la tercera capa del pastel de las tecnologías web estándar, dos de las cuales (HTML y CSS) hemos cubierto con mucho más detalle en otras partes del Área de aprendizaje.
